# Sprint 003 Requirements: Admin Onboarding Minimal

## Goal

Enable a school to be set up with the minimum data needed for the MVP.

## In Scope

- Create school.
- Import or create students.
- Create classes.
- Assign students to classes.
- Assign Guru/Wali Kelas to classes.
- Create parent invitations or link codes.
- Link parents to students.

## Out of Scope

- Full TU module.
- Payments.
- Documents/letters.
- Staff attendance.
- Institutional calendar.

## Notes

Detailed blueprint waits until Sprint 002 confirms the foundation data/auth implementation.
