# Sprint 004 Requirements: Guru Daily Loop

## Goal

Build the teacher daily loop around Papan Pagi, morning attendance, parent messages, and students needing attention.

## In Scope

- Papan Pagi ordered as:
  1. Status Absensi Hari Ini
  2. Pesan Ortu Belum Dibalas
  3. Siswa Perlu Perhatian
  4. Jadwal Mengajar Hari Ini
- Morning attendance statuses: Hadir, Sakit, Izin, Alpa, Terlambat.
- Configurable attendance cutoff from `school_settings`.
- Attendance notifications for Sakit, Izin, Alpa, Terlambat.
- Same-day edits and post-day correction flow.
- Parent messages without formal SLA.

## Out of Scope

- Parent PWA polish.
- Full grade workflow.
- Full notes workflow.
- Principal analytics.
