# Sprint 004 Acceptance Criteria

Sprint 004 is complete when:

- Guru/Wali Kelas can open Papan Pagi.
- Attendance can be completed for a class.
- Attendance status supports Hadir, Sakit, Izin, Alpa, and Terlambat.
- Attendance cutoff status is visible.
- Non-present/late attendance creates notification records.
- Same-day edit works.
- Post-day correction requires a reason.
- Unreplied parent messages appear on Papan Pagi.
- Siswa Perlu Perhatian includes Alpa today, Terlambat today, or status Perhatian/Kritis.
